## command
- C-b c
  - create new window

- C-b space
  - copy-mode (move vim mode 'hjkl'. select: space. copy: Enter)

- C-b %
  - split vertical panel.

- C-b "
  - split Horizontal panel.

- C-b q
  - move panel.

- C-b x
  - delete active panel.

- C-b &
  - delete active window.
